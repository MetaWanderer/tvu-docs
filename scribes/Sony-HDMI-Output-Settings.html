📺 Camera Setup
Sony HDMI Output Settings
Correct HDMI resolution configuration for TVU IRL streaming
⏱ Time: ~3 minutes
🎯 Required for: TVU Pack users
Compatible Cameras
Sony FX3
Sony FX6
Sony A7 IV
Sony A7S III
Most Sony mirrorless after 2021
📡
TVU Pack users — this setting is critical
The TVU Pack inherits its streaming resolution directly from the camera's HDMI output setting. Almost all IRL setups run at 1080p — this must match your streaming platform (Twitch, YouTube, Kick).
Walkthrough
Step-by-Step Setup
1
Open the Camera Menu
Press the physical MENU button on the camera body to open the main menu.
2
Navigate to External Output
Follow this menu path to reach the HDMI output settings:
Setup › External Output › HDMI Output Settings
3
Set HDMI Output Resolution to 1080p
Inside HDMI Output Settings, set the resolution. This is almost always 1080p for IRL streaming.
HDMI Output ResolutionUnder HDMI Output Settings
1080p
⚠️
The TVU Pack will stream at whatever resolution the camera outputs over HDMI. If this is set wrong, your stream resolution will be wrong.
4
Disable HDMI Info Display
This removes all on-screen overlays (menus, indicators) from the HDMI output so your stream receives a clean video signal.
HDMI Info. DisplayUnder External Output
Off
✅
With this off, your HDMI output will be a clean picture with no camera UI elements visible in your stream.
✅ Quick Reference — Settings Checklist
✓MENU → Setup → External Output → HDMI Output Settings
✓HDMI Output Resolution → 1080p
✓HDMI Info. Display → Off
✓CTRL FOR HDMI → Off (recommended)
