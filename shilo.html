Style & Settings

Drew — Sony FX3

4K 23.98fps 10-bit 4:2:2 S-Gamut3.Cine / S-Log3 f/2.8 – 4 1/48 shutter ISO 800 or 12,800 24-70mm + ND
  • Expose with DisplayLUT + histogram — 1.3–2.0 stops over
  • Base ISOs only — 800 outdoor, 12,800 low light (gym/restaurant)
  • Mic talent before streaming — clean in-camera audio all day

Shooting Approach

  • Moments first — stay ready, let things happen naturally
  • Shoot for the edit — wides, mediums, close-ups. Hold each shot 10+ sec
  • Action + reaction — talent doing = main shot, what they see = cutaway
  • Frame for vertical — center subjects for 9:16 crop-ability
  • Capture stream POV — phone screens, chat reactions, TVU pack UI, pack setup
  • Gym specifics — murals, car, neon lights, equipment. Shoot the vibe.
  • Table tennis — both sides of the table if second angle available via GO
  • Drive segments — pull from the stream + get window reflections, speed, city blur

Slow Motion

  • S&Q mode at 120fps, shutter 1/250
  • Table tennis volleys, boxing combos, gear shots, Miami environment
  • Only when comfortable — don't miss the moment chasing slow-mo
