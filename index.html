Camera
Style & Settings

Both Cameras — FX3 Settings

4K 23.98fps S-Gamut3.Cine / S-Log3 f/2.8 – 4 1/48 shutter ISO 800 or 12,800 24-70mm + ND Hollyland mics
  • Expose with DisplayLUT + histogram — 1.3–2.0 stops over
  • Mic talent before streaming — clean in-camera audio all day
  • Base ISOs only — 800 (daylight) or 12,800 (low light)

Shooting Approach

  • Moments first — stay ready, let things happen naturally
  • Shoot for the edit — wides, mediums, close-ups. Hold each shot 10+ sec
  • Action + reaction — talent doing = main shot, what they see = cutaway
  • Shoot & move — follow talent but change angles, find extra visuals
  • B-roll between segments — streets, food, gear details, environment
  • Capture stream POV — phone screens, chat reactions, GO app UI
  • Frame for vertical — center subjects for 9:16 crop-ability

Bonus — Slow Motion Hero Shots

  • S&Q mode at 120fps, shutter 1/250
  • Cinematic reveals, gear shots, LA environment
  • Only when comfortable — don't miss the moment chasing slow-mo
