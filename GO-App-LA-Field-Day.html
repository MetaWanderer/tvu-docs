01 — Purpose
Why We're Doing This
This is not a commercial shoot.
This is a creator-first field test + content day. We want to let Keem use GO naturally, capture real moments, and show how mobile streaming is evolving — creating content that works for both audiences (TVU + creators).

Real-World Usage

Let Keem use GO the way he normally would — no scripts, no feature lists, just authentic IRL streaming.

Capture Moments

Cinematic BTS alongside live streams. The stream comes first — we film the story around it.

Show Evolution

Demonstrate how mobile streaming is evolving through real use — not a product demo.

Dual Audience

Content that resonates with both TVU's audience and the creator communities of Keem & Jake.

Organic
Unscripted

03 — Segment Breakdown
What We're Shooting

Each segment is designed to stand alone while building a full narrative arc for the day.

Segment 1
Keem Solo — Day in the Life + Tech + Walk Test
Keem
45–75 min

What We Capture

  • How Keem actually prepares for IRL (gear, mindset, routine)
  • GO App walkthrough in context — not a feature list
  • Disconnect Protection demo on alt channel
  • Toolkit integration demo (positioned as "works with your existing stack")
  • Real-world walk test

Needs

  • TVU backpack for Keem
  • Camera operator (Luca preferred)
  • Talking points for GO features (not a script)
  • Alt channel set up for Disconnect Protection demo

Segment 1B
Keem Solo IRL Content — Pokemon / Cards / Store
Keem
30–60 min

What We Capture

  • Picture-in-Picture mode demonstration

Segment 2
Jake Solo — Explore LA / First Impressions
Jake
30–60 min

What We Capture

Segment 4
Crossover Challenge — $25 IRL Streaming Challenge
Keem Jake
30–45 min

  • Keem and Jake split up for 30 minutes

Backup Ideas

Goal: Fun, competitive energy. Split POV content creates natural tension and a satisfying payoff at the reveal.
